# Seed Thoughts & Synchronicity — Site

## Project Context

Website for Joan Woods Jackson's book *Seed Thoughts & Synchronicity: A Guide to Your Inner Wisdom*.

**About Joan:** Joan Woods Jackson, 80 years old, not tech-savvy. She wrote the book and wants a simple, beautiful website to promote it.

**Contact:** jacksonjo@aol.com

**Domains:**
- seedthoughtsandsynchronicity.com (primary)
- yourwildmagic.com (secondary)

## Technical Stack

- **Static HTML/CSS/JS** — no frameworks, no build tools
- **Hosted on GitHub Pages** with GitHub Actions deployment
- **No npm, no bundlers** — keep it simple

## Design Direction

- **Colors:** Warm golds/ambers, muted purples/mauves, soft cream, earth tones
- **Typography:** Serif fonts (Georgia, Times New Roman)
- **Feel:** Warm, inviting, spiritual, accessible to older readers

## Development

To preview locally, open `index.html` in a browser. No build step required.

Pushing to `main` triggers automatic deployment to GitHub Pages.

## Checks

This is a static site — there are no typecheck, lint, test, or build commands.
Validate HTML manually or with an HTML validator if needed.
