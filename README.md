# Seed Thoughts & Synchronicity

A website for Joan Woods Jackson's book *Seed Thoughts & Synchronicity: A Guide to Your Inner Wisdom*.

## Development

Static HTML/CSS/JS site hosted on GitHub Pages. No frameworks or build tools required.

- **Primary domain:** seedthoughtsandsynchronicity.com
- **GitHub Pages:** https://gilbetrar.github.io/seed-thoughts-site/

## Deployment

Push to `main` to deploy automatically via GitHub Actions.
